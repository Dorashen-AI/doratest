口袋精灵读卡器 WE-PK23是海南世电科技推出的 2.4GHz 无线蓝牙仓储 / 人员智能管理设备，支持电脑 USB 串口与手机蓝牙双连接，单机可管理1000 个资料、32 个物品分类，可同时呼叫80 个电子标签、最远读卡 50 米、蓝牙传输 30 米，内置750mAh 锂电池续航超 15 小时；搭配电脑管理软件和口袋精灵手机 APP，可实现物品出入库、库存盘点、人员管理、标签声光找物、数据备份还原等功能，适配一维 / 二维码扫码枪，适配多类声光电子标签，满足仓库物资、人员实时管控与快速查找需求。
一、产品概述
口袋精灵读卡器 WE-PK23集传统仓库操作与智能化管理于一体，可完成物品登记、出入库、明细查询、人员 / 物品快速查找、实时盘点；数据仅保存在读卡器本地，电脑与 APP 不存储，保障数据安全，无需多设备同步；一台设备可管理1000 种资料，留存至少一年出入库明细与人员绑定关系。
支持电脑 USB 有线、手机蓝牙无线双连接，电脑做日常管理、手机移动找货找人，适配仓库常态化管理场景。
二、主要核心特性
管理上限：支持1000 个资料管理、32 个物品分类，单物料可归属多个分类。
日志留存：自动保存至少 1 年出入库及物品操作日志。
外设兼容：支持一维码、二维码扫码枪完成入库、出库、查询操作。
标签呼叫：可同时呼叫80 个电子标签，声光提醒找物，最远50 米读卡距离。
续航能力：标配750mAh 锂电池，连续工作15 小时以上。
蓝牙性能：2.4GHz 发射功率 8dBm，无线传输最远30 米。
外观材质：ABS 工程塑料，尺寸97mm×50mm×19mm，便携易带。
配套软件：免费电脑管理软件 + 手机口袋精灵 APP，全流程管控。
