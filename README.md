## 下载与安装步骤
1.  访问 [Releases 页面](https://github.com/Dorashen-AI/doratest/releases)，下载你需要的文件
2.  安装前请核对文件的 `sha256` 校验码，确保文件完整未被篡改
3.  安卓 APP 安装请开启「未知来源应用安装」权限
4.  电脑端软件请解压后按说明安装，首次使用需安装对应串口驱动
